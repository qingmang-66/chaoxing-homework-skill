# 浏览器自动化：网页作业/表单填写技能

通过 CDP 控制浏览器，自动填写在线作业、问卷、表单。

## 功能

- 批量遍历课程，检查未完成作业
- 自动填写单选题、多选题、判断题
- 自动填写填空题、名词解释、简答题（UEditor 富文本编辑器）
- 支持 iframe 跨域操作
- 自动保存（默认只保存，不提交）

## 核心流程

```
连接 CDP → 检查当前页面状态 → 遍历课程 → 读取作业列表
→ 点击未完成作业（新标签页打开）→ 填写答案 → 暂时保存
```

## 已验证的关键发现

- `frame_selector + ref + click` 可以成功触发导航，但会在**新标签页**中打开
- 验证导航成功：`tabs list` 检查所有标签页，找 URL 含 `/work/dowork` 的新标签页
- `snapshot` 能跨域读取 iframe 完整渲染结果，`eval` 只能访问外层 iframe 的同域 DOM
- 默认只保存，不提交（除非用户明确同意）
